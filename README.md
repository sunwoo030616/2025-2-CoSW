## 📖 프로젝트 소개

경찰청에서 운영하는 분실물 통합 사이트 **LOST112**가 복잡한 UI와 검색 방식으로 이용하기 어렵다는 후기가 많다는 점에 착안하여, 경찰청 공공데이터를 활용해 **누구나 쉽고 편하게 분실물을 찾을 수 있는 챗봇 서비스**를 기획·개발했습니다.

## 💡 기획 배경

- 기존 LOST112 사이트는 복잡한 카테고리 분류와 검색 UI로 인해 실제 이용자들의 불편 후기가 다수 존재
- 사용자가 직접 카테고리를 탐색하지 않아도, 대화하듯 질문하면 원하는 정보를 찾을 수 있는 서비스의 필요성을 느껴 기획
- 실제 유실물을 잃어버렸던 동기들의 인터뷰를 통해 사용자가 가장 필요로 하는 기능을 파악하고 서비스 흐름 설계

## ✋ 담당 역할

- **기획**: 서비스 컨셉 및 사용자 시나리오 설계, 경찰청 공공데이터 활용 방안 기획
- **디자인**: Figma 기반 UI/UX 디자인
- **프론트엔드**: HTML/CSS/JavaScript를 활용한 챗봇 인터페이스 구현
